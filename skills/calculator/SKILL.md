# Calculator Skill

## Description

A simple calculator skill that performs basic arithmetic operations. This skill provides tools for addition, subtraction, multiplication, and division.

## Capabilities

- **add**: Add two numbers together
- **subtract**: Subtract one number from another
- **multiply**: Multiply two numbers
- **divide**: Divide one number by another

## Usage

The calculator skill can be used to perform mathematical operations within conversations. The agent can automatically detect when calculations are needed and use the appropriate tool.

## Tools

### add(a: number, b: number)
Adds two numbers and returns the result.

### subtract(a: number, b: number)
Subtracts the second number from the first and returns the result.

### multiply(a: number, b: number)
Multiplies two numbers and returns the result.

### divide(a: number, b: number)
Divides the first number by the second and returns the result.
