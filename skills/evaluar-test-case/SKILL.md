# Evaluar Especificación de Test Case
Este skill permite transformar descripciones de prueba informales en especificaciones de alta calidad. 

## Casos de Uso
- Cuando el usuario pega un texto de un caso de prueba.
- Cuando el usuario proporciona un ID de issue de GitLab (ej. "Revisa el issue #502").
